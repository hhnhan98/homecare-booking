# HomeCare Booking System

Hệ thống đặt lịch khám ngoại trú tại nhà — Demo tài liệu cho môn "Công cụ và phát triển phần mềm"

## Mục tiêu

- Nộp báo cáo URD, thiết kế UI/UX (Figma), timeline, và danh sách issues theo yêu cầu môn học.
- (Tùy chọn) Docker + Azure để demo.

## Cấu trúc repo

- /backend — code backend (nếu có)
- /frontend — code frontend (nếu có)
- /docs — URD, ERD, wireframes, issues CSV
- /.github — CI (nếu có)
